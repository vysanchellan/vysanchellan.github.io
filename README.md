# vysanchellan.github.io

Personal website + booking demos for **Vysan Studio** — premium booking-first websites for service businesses in KwaZulu-Natal, South Africa.

Built as a static GitHub Pages site (no build step). Just open `index.html` in a browser, or visit:

- **Live**: https://vysanchellan.github.io
- **Local preview**: `python3 -m http.server 8080` from the repo root, then open http://localhost:8080

## Pages

| Page | What it shows |
| --- | --- |
| `index.html` | Landing page: hero, work showcase, services, process, pricing, testimonials, FAQ, contact form. |
| `optometrist.html` | Demo: booking-driven optometrist site with interactive calendar + AI concierge. |
| `pharmacy.html` | Demo: pharmacy site with vaccination/refill bookings + AI assistant. |
| `clinic.html` | Demo: private-clinic site with specialist consultations + concierge AI. |

## Stack

Plain HTML / CSS / vanilla JS. No build, no framework. Google Fonts (Inter, Fraunces, JetBrains Mono) loaded via CDN.
